# MVP - Análise de Dados e Boas Práticas (PUC-Rio)

**Aluno:** Philippe Lima  
**Tema:** Gestão de Risco em Obras com BIM e Inteligência Artificial

## Sobre o Projeto
Este repositório contém o trabalho final (MVP) da sprint de Análise de Dados e Boas Práticas da pós-graduação da PUC-Rio. 

O objetivo do projeto é analisar o "BIM-AI Integrated Dataset" para identificar quais variáveis de execução de obra (como custos, atrasos e alocação de recursos) mais impactam o nível de risco de um projeto de construção civil.

## Arquivos do Repositório
* `bim_ai_dataset.csv`: Base de dados original utilizada na análise.
* `MVP_Analise_Dados_Philippe.ipynb`: Notebook em Python (Google Colab) contendo todo o código documentado, desde a importação dos dados até a análise exploratória e o pré-processamento.
